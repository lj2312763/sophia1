AI文本服务接口说明⽂档

更新时间：2020-03-10



简介

自然语言处理是为各类企业及开发者提供的用于文本分析及挖掘的核心工具，已经广泛应用在电商、文化娱乐、金融、物流等行业客户的

多项业务中。自然语言处理API可帮助用户搭建内容搜索、内容推荐、舆情识别及分析、文本结构化、对话机器人等智能产品，也能够通

过合作，定制个性化的解决方案。

  • 接口类型：通过 REST API 的方式提供的通用的 HTTP 接口。适用于任意操作系统，任意编程语言

  • 接口参数：具体详见接口请求说明。



调用流程

  1. 通过sophia AI开放注册进行调用。

  2. 返回识别结果：识别结果会即刻返回，采用 JSON 格式封装，如果调用成功，识别结果放在 JSON的“data”字段中，统一采用 utf-8 方式

    编码。具体详见返回说明。

请求列表


接口名称 功能描述
分词和词性标注（Segment & Part of Speech, POS） 对中文文本进行分词和词性标注
句法分析（Syntactic Parser） 用以获取句⼦的句法结构
句式识别（Sentence Type） ⽤来识别句式，如特殊疑问句、祈使句等
依存句法分析（Dependency Parser, DP） 给出句⼦中词汇之间的依存关系
关键词抽取（Keyword） ⽤来抽取句⼦中的关键词
命名实体识别（Named Entity Recognition, NER） 识别句⼦中的命名实体
语义相似度（SSM） ⽀持词和句⼦的语义相似度的计算
句⼦边界识别（SBD） 根据句⼦标点和语义进⾏断句
