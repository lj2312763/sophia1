律师查询接口

更新时间：2020-03-10



简介

律师查询接口通过直接输入律师姓名，选择想要查询的律师。适用于律师基本信息（包括律所，代理案件的信息）、律师案由分布、律师

法院层级分布、律师案件类型分布、律师案件时间分布、律师诉讼阶段等数据统计查询场景。

  • 接口类型：通过 REST API 的方式提供的通用的 HTTP 接口。适用于任意操作系统，任意编程语言

  • 接口参数：输入律师姓名即可调用律师查询，具体详见请求说明。


调用流程

  1. 通过sophia AI开放注册进行调用。

  2. 返回识别结果：识别结果会即刻返回，采用 JSON 格式封装，如果调用成功，识别结果放在 JSON的“data”字段中，统一采用 utf-8 方式

    编码。具体详见返回说明。


请求说明

请求示例

HTTP 方法：POST

请求URL： https://www.yzsophia.com/insight/lawQuery/lawyer_search


请求参数

参数 是否必选 类型 可选值范围 说明
keyword 是 string - 关键词，律师姓名，例如：“小明”

pageNo

是 string 正整数范围 页号
pageSize 是 string 正整数范围 页大小



返回参数

字段 是否必选 类型 说明
fullName 是 string 律师姓名
firmName 是 string 所属律所
id 是 string 律师id
mount 是 string 案件数
lawFirmId 是 string 律所ID
bizLawFirmReturnListModels 是 List 律师案件集合
