地理海拔服务接口说明

更新时间：2020-03-10



简介

查询特定省份市区下的地点海拔数据。

  • 请求说明：通过 REST API 的方式提供的通用的 HTTP 接口；Json格式Post方式；适用于任意操作系统，任意编程语言。


调用流程

  1. 通过sophia AI开放注册进行调用。

  2. 返回识别结果：识别结果会即刻返回，采用 JSON 格式封装，如果调用成功，识别结果放在 JSON的“data”字段中，统一采用 utf-8 方式

    编码。具体详见返回说明。

请求说明

请求示例

HTTP 方法：POST

请求URL： http://


请求参数

参数 是否必选 类型 可选值范围 说明
provinceName 否 string - 省份
cityName 否 date - 市


请求示例 

{
  "provinceName": "河北省"，
  "cityName": "衡水市"
}


返回参数

字段 是否必选 类型 说明
dataid 是 string id
provinceName 是 string 省份名称
lng 是 string 经度
lat 是 string 维度
altitude 是 string 海拔
address 是 string 地址
description 是 string 描述
provinceCode 是 string 省份code
cityName 是 string 城市名称
cityCode 是 string 城市code
