旅游人流量接口说明

更新时间：2020-03-10



简介

查询特定景区某一时间段内的人流量值。

  • 请求说明：通过 REST API 的方式提供的通用的 HTTP 接口；Json格式Post方式；适用于任意操作系统，任意编程语言。


调用流程

  1. 通过sophia AI开放注册进行调用。

  2. 返回识别结果：识别结果会即刻返回，采用 JSON 格式封装，如果调用成功，识别结果放在 JSON的“data”字段中，统一采用 utf-8 方式

    编码。具体详见返回说明。

请求说明

请求示例

HTTP 方法：POST

请求URL： http://


请求参数

参数 是否必选 类型 可选值范围 说明
scenic_spot 否 string - 景点名称
Start_time 否 date - 开始时间
End_time 否 date - 截止时间


请求示例 

{
  "scenic_spot": "北京欢乐谷"，
"Start_time": "2019-03-08"，
"End_time": "2019-03-09"，

}
