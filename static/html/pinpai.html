品牌数据检索接口

更新时间：2020-03-10



简介

品牌数据检索接口通过直接输入商品的品类，平台，店铺，评价维度等查询品牌商品的详情以及评价统计信息。适用于查询商品品牌信息，

纵向进行商品的评价统计以及分析决策等场景。

  • 接口类型：通过 REST API 的方式提供的通用的 HTTP 接口。适用于任意操作系统，任意编程语言。

  • 接口参数：输入商品的品类，平台，店铺，评价维度即可调用接口查询品牌商品的详情以及评价统计信息，具体详见请求说明。


调用流程

  1. 通过sophia AI开放注册进行调用。

  2. 返回识别结果：识别结果会即刻返回，采用 JSON 格式封装，如果调用成功，识别结果放在 JSON的“data”字段中，统一采用 utf-8 方式

    编码。具体详见返回说明。

请求说明

请求示例

HTTP 方法：POST

请求URL： https://www.yzsophia.com/insight/product/listProductInsight


请求参数

参数 是否必选 类型 可选值范围 说明
productCategory 否 string - 品类
productBrand 否 string - 品牌
platForm 否 string - 平台
store 否 string - 店铺
productModel 否 string - 系列
evaluateCategory 否 string 0-中性 1-正面 2-负面 评论情感值
evaluateDimension 否 string 产品、运营、物流、 其他 评论维度：产品 运营 物流 其他
beginDate 否 string - 评论时间-起始时间，格式：2020-03-08 13:00:00
endDate 否 string - 评论时间-截止时间，格式：2020-03-08 23:00:00
orderField 否 string - 排序字段
orderType 否 string asc，desc 排序类型，例如：升序=asc, 降序=desc
pageIndex 是 string 正整数 页码
pageSize 是 string 正整数 页大小


返回参数

字段 是否必选 类型 说明
productCategory 是 string 品类
productBrand 是 string 品牌
productPlatform 是 string 平台
productModel 是 string 型号
productStore 是 string 店铺
favorableRate 是 string 好评率
evaluateCategory 是 string 评价类型(0=好评，1=中评，2=差评)
userNick 是 string 用户昵称
evaluateInfo 是 string 评论详情
evaluateTime 是 string 评论时间
evaluateDimensionOne 是 string 评论维度1
evaluateDimensionTwo 是 string 评论维度2
evaluateDimensionThree 是 string 评论维度3
evaluateDimensionFire 是 string 评论维度4
